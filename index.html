How to Use CracTrac

Working Offline: On your mobile device or desktop, use the app to view the site overview, select equipment, and add cracks by tapping on equipment drawings. Each crack is recorded with a photo, note, and severity rating. Markers are positioned based on the imageâ€™s natural dimensions.

Export/Import: Use the Export Data button to download a ZIP file containing your crack data as JSON. Later, use the Import Data button on your desktop to load the data. Equipment with no data will show zero cracks.

Generate Report: Click the Generate Report button to create a PDF report. The report includes a title page, an overview page with the site image, a summary table, and detailed crack information (with equipment drawings annotated with crack markers). The PDF is automatically downloaded.
